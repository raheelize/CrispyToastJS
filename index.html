Introduction

The CrispyToast library provides a lightweight solution for displaying colorful and customizable toast messages in your web application. Toast messages are small, unobtrusive notifications that appear at the bottom of the screen, providing feedback to the user without interrupting their workflow.

Usage

Include Below Mentioned CDNs

       https://raheelize.github.io/CrispyToastJS/crispy-toast.css //FOR CSS
       https://raheelize.github.io/CrispyToastJS/crispy-toast.js  //FOR JS

To use the CrispyToast library, simply include the crispy-toast.js script and crispy-toast.css stylesheet in your HTML file. Then, you can create different types of toast messages using the provided functions: 


        // Display a simple toast message
        CrispyToast.message('Hi, Enjoy Using Crispy Toast!.');

        // Display a success toast message
        CrispyToast.success('Success! Your action was completed.');

        // Display an info toast message
        CrispyToast.info('Info: This is an informational message.');

        // Display a warning toast message
        CrispyToast.warning('Warning: Something might not be right.');

        // Display an error toast message
        CrispyToast.error('Error: Oops! An error occurred.');

Options

You can customize the behavior of the toast messages by providing optional options: 


        // Display a success toast with a longer timeout
        CrispyToast.success('Success!', { timeout: 5000 });

        // Display a warning toast positioned at the top-left corner
        CrispyToast.warning('Warning!', { position: 'top-left' });

        // Display a error toast positioned at the bottom-left corner and a longer timeout
        CrispyToast.error('Error!', { position: 'bottom-left',timeout: 8000  });

Styling

The toast messages are styled using the crispy-toast.css stylesheet. You can modify this stylesheet to customize the appearance of the toasts according to your project's design.

Conclusion

With the CrispyToast library, you can easily add eye-catching and informative toast messages to your web application, improving user experience and providing timely feedback.
